Phone Identifier Test

This page shows what the phone identifiers should look like when bid data is returned from the server.

Item: Soccer Ball

Current Bid: $75

Next minimum bid: $80

***-4567

Item: Basketball (No bids yet)

Current Bid: $200 (starting)

Next minimum bid: $205

Item: Artwork Print

Current Bid: $150

Next minimum bid: $160

***-8901

Simulated Server Response:

{
  "Soccer Ball": {
    "bidAmount": 75,
    "phone": "555-123-4567",
    "fullName": "John Doe",
    "timestamp": "2025-10-19T10:30:00"
  },
  "Artwork Print": {
    "bidAmount": 150,
    "phone": "(555) 987-8901",
    "fullName": "Jane Smith",
    "timestamp": "2025-10-19T11:45:00"
  }
}

Expected behavior:

  • ✅ Items with bids should show "***-XXXX" where XXXX is the last 4 digits of the bidder's phone
  • ✅ Items without bids yet should NOT show any phone identifier
  • ✅ The phone identifier should appear above the bid buttons
  • ✅ It should have a gray background and rounded corners
